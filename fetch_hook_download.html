Hook for downloading files

Usage

fetch_hook_download(furl, fread)

Arguments

furl
Function to convert key, namespace into a URL. Takes scalar strings for key and namespace and returns a scalar string URL. Depending on the application you may not need the namespace argument but your function must accept it even if it just ignores it.
fread
Function for converting filename into an R object. filename will be a scalar character and be a filename that will exist on the system. fread can return anything and may throw an error if the given file was not in a valid format. Functions read.csv, readRDS can be used here as-is (though for the former consider function(filename) read.csv(filename, stringsAsFactors=FALSE)).

Description

Storr hook for downloading files as an external resource. This might be useful for other, more general projects. Just depend on this package (which drags in curl) and use this as the fetch_hook argument to storr_external.
